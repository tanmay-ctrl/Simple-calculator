# Simple Calculator

A clean, responsive, and fully functional simple calculator built from scratch using HTML, CSS, and vanilla JavaScript.

🟢 [View Live Demo](https://tanmay-ctrl.github.io/Simple-calculator/)

## Features

* **Basic Arithmetic:** Supports addition, subtraction, multiplication, and division.
* **Responsive Design:** Adapts smoothly to both mobile and desktop screens.
* **Error Handling:** Handles edge cases like clearing the display and preventing division by zero.

## Tech Stack

* HTML5, CSS3, JavaScript (ES6)
* Libraries: None (Vanilla JS)
